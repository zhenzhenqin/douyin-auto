# 🔥 抖音自动续火花助手 (绿色免安装版)

这是一个基于 Node.js + Playwright 实现的自动化工具，旨在帮助用户自动维持抖音好友火花，同时通过后台运行模式防止刷视频上瘾。

## ✨ 功能特点

- **✅ 零门槛使用**：内置 Node.js 环境，解压即用，无需安装配置。
- **📺 后台静默运行**：脚本在后台运行，不弹出浏览器窗口，不影响你使用电脑。
- **⏰ 每日定时任务**：一键设置 Windows 定时任务，每天早上自动唤醒电脑执行。
- **👥 批量群发**：支持添加多个好友，脚本会自动搜索并发送。
- **🛡️ 智能防风控**：模拟真人操作间隔，支持随机启动时间，降低风险。
- **🔌 智能网络检测**：休眠唤醒后自动等待网络连接，防止报错。

## 📦 如何下载

请点击右侧 **Releases** 下载最新版本的压缩包（或者直接点击仓库绿色的 **Code -> Download ZIP**）。

## 🚀 使用教程 (必读)

**第一步：解压**
下载后，请务必将压缩包**解压**到一个文件夹中（不要在压缩包内直接双击！）。

**第二步：初始化环境 (仅首次)**
双击运行文件夹内的 **`首次使用请点我.bat`**。

> ⏳ 这步会自动下载浏览器驱动，大概需要 1-2 分钟，请耐心等待直到窗口提示完成。

**第三步：启动控制台**
双击运行 **`mjc-start.bat`**。
浏览器会自动打开控制面板界面 (http://localhost:3000)。

## ⚙️ 配置与运行

1.  **添加好友**：在网页输入框填写好友的**备注名**或**昵称**（推荐设置简短的备注），点击添加。
2.  **扫码登录**：点击“扫码登录”按钮，拿出手机抖音扫码，登录成功后脚本会自动保存凭证。
3.  **测试运行**：点击“立即群发一次”，观察脚本是否正常工作。
4.  **开启自动任务**：设置好时间（如 08:00），点击“更新系统定时任务”。
    
    > ✅ 设置成功后，哪怕你关闭了所有窗口，电脑只要开着（或休眠），每天都会自动执行。

## ⚠️ 风险提示与免责声明

1.  **封号风险**：任何自动化工具都有被平台风控的风险。建议**好友数量控制在 20 人以内**，且不要作为营销工具滥用。
2.  **仅供学习**：本项目仅供编程学习与技术交流使用，请勿用于非法用途。
3.  **账号安全**：登录凭证（`auth.json`）保存在你本地电脑，请勿将整个文件夹发送给陌生人，否则可能导致账号被盗。

## 🛠️ 常见问题

- **Q: 点了启动没反应？**
  A: 请尝试右键 `启动程序.bat` 选择“以管理员身份运行”。
- **Q: 报错找不到浏览器？**
  A: 请确保你执行了第二步的初始化脚本。
- **Q: 任务没自动跑？**
  A: 检查你的电脑在设定时间是否处于关机状态（关机无法运行，睡眠/休眠可以）。
